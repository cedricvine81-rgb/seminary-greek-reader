# Instructor Workflow

## 1. Create an Account

Go to `/auth/sign-up`, select **Instructor**, fill in your details.

## 2. Create a Course

From the Instructor Dashboard → **Courses** → **New Course**.

Fill in:
- Course name (e.g., "Beginning Greek I — Fall 2025")
- Institution
- Level: Beginning or Intermediate
- Start / End dates

## 3. Invite Students

Students sign up independently and enroll via their Student Dashboard. (Email invite flow is planned for a future release.)

## 4. Create Assignments

Go to **Assignments** → **New Assignment**.

- Choose assignment type: Vocabulary Quiz, Morphology Quiz, or Translation Exercise
- Set week number, due date, and level
- Optionally provide a biblical reference (e.g., "John 1:1-5")
- Set the number of questions

Questions are **auto-generated** based on the level and type:
- Vocabulary quizzes pull from the NT frequency lists
- Morphology quizzes pull from parsed verse words in the database

## 5. Upload Materials

Go to **Materials** → **Upload** to add course notes, handouts, or links for each week.

## 6. View Progress and Reports

- **Students** page: see enrollment counts and average scores per student
- **Reports** page: generate a semester report and export as CSV
