# Copyright Notes

## Biblical Texts

All Greek texts included in this application are public domain.

- **Septuagint (LXX)**: The Greek Old Testament is in the public domain. The sample text in `prisma/seed-data/sample-lxx.json` represents the received LXX text, which has been freely available since antiquity.
- **Greek New Testament (GNT)**: The underlying Greek text (Byzantine/Majority Text tradition) is in the public domain. The sample text in `prisma/seed-data/sample-gnt.json` follows the public domain tradition.

> For a full, freely licensed digital text, see:
> - **SBLGNT** (Society of Biblical Literature GNT) — free for non-commercial use; see their license.
> - **MorphGNT** — CC-BY-SA — https://github.com/morphgnt/sblgnt
> - **LXX** from CCAT/OTA — freely available academic versions.

## Lexical Data

- Vocabulary glosses and frequency data in the seed files are original editorial work based on well-established scholarly consensus and are not reproduced from any single copyrighted lexicon.
- No content from BDAG, HALOT, LSJ (Oxford editions), or other copyrighted lexicons has been reproduced.

## Grammar Category Labels

- The syntax categories in `prisma/seed-data/syntax-categories.json` and `src/data/syntax-categories.ts` consist of short descriptive labels.
- These labels describe standard grammatical categories that are common knowledge in NT Greek scholarship.
- No text from *Greek Grammar Beyond the Basics* (Wallace) or any other copyrighted grammar has been reproduced. Only short, conventional category names are used.

## Summary

This application contains no copyrighted biblical text, lexicon content, or grammar prose. All content is either:
1. Public domain
2. Original editorial work
3. Short conventional category labels (not protectable expression)
