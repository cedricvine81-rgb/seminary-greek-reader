# Greek Fonts

Place Greek-capable web fonts here. Recommended free/open options:

- **Gentium Plus** — SIL Open Font License — https://software.sil.org/gentium/
- **GFS Didot** — OFL — https://www.greekfontsociety-gfs.gr/

After downloading, add them to `src/app/globals.css` with `@font-face` declarations.

The app's Tailwind config already references `font-greek: ['Gentium Plus', 'GFS Didot', 'serif']`.
